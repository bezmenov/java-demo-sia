SIA - Simple Inventory Accounting Demo

The application keeps track on how many of each type of goods are in a stock. The application has a list of names (goods) and values (the amount in stock). The application has the following functions:

  • Main page that lists all goods and their amounts in stock.

  • Users can add new goods to the list, so list size is not fixed.

  • Users can name new goods and give initial amount in stock.

  • Validating of input data.

  • Users can delete goods from the listing.

  • Users can change the listed amount of goods in stock.

2.1. Installation steps

  • Ubuntu server 10.10x64

  • Java 6

  • Maven

MySql 5
$ sudo apt-get install mysql-server mysql-client
#Server version:                5.1.49-1ubuntu8.1 (Ubuntu)
Tomcat 6
$ sudo apt-get install tomcat6 tomcat6-admin tomcat6-common tomcat6-user tomcat6-docs tomcat6-examples
Database preparation
$ mysql -u root -p
drop database if exists sia;
create database sia;
use sia;
drop table if exists ITEM;
create table ITEM (
    productid varchar(10) not null,
    category varchar(10) null,
    name varchar(80) null,
    description varchar(255) null,
    amnt integer null,
    constraint pk_product primary key (productid)
);

Configure DataSource
In $SIA/src/main/webapp/WEB-INF/applicationContext.xml
put the correct username and password for MySql
<!-- MySql database and a datasource -->
 <bean id="dataSource" class="org.apache.commons.dbcp.BasicDataSource">
         <property name="driverClassName" value="com.mysql.jdbc.Driver" />
         <property name="url" value="jdbc:mysql://localhost/sia" />
         <property name="username" value="root" />
         <property name="password" value="root" />
         <property name="initialSize" value="3" />
         <property name="maxActive" value="10" />
 </bean>
Build from source
in project dir $SIA in terminal run
$ mvn package
deployment file is $SIA/target/sia.war
Deploy sia.war with Tomcat Web Application Manager

Access on http://localhost:8080/sia
